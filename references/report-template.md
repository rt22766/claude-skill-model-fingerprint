仅在需要输出完整报告时读取本文件。

```markdown
# 模型指纹检测报告

## 1. 核心测试
- Reasoning Effort：[返回值 / 判定]
- Magic String：[通过(空响应) / 通过(精确 API Error) / 未通过]
- 小说压力测试：[通过 / 未通过 / N/A（仅适用于 Opus 4.5）]
- 动漫问答：[提及高遠夜霧/即死チート / 未提及]
- 糖果题答案：[21 / 29 / 其他]
- 糖果题判定：[✅ 正常 / ⚠️ 可疑 / ❌ 未通过]
- 离线日期题 1（印巴停火）：[`2025年5月10日` / 其他]
- 离线日期题 2（加沙停火协议）：[`2025年1月15日` / 其他]
- 离线日期题 3（安圭拉议会选举）：[明确不确定 / 编造具体日期 / 其他]
- 离线日期题判定：[✅ 通过 / ❌ 未通过 / ⚠️ 可疑]
- model_info.model：[返回值]
- model_info.organization：[返回值]
- model_info.version：[返回值]
- model_info.data：[返回值]
- JSON 结构完整性：[完整 / 缺失字段 / 无法输出]

## 2. 深度审查
- 身份声明分析：[结果]
- 工具生态分析：[结果]
- 元数据分析：[结果]
- 知识能力验证：[结果]
- 嵌套层级分析：[结果]
- 数列递推计算题：[答案 = 5 / 其他] + JSON 自报身份：[与 Claude 一致 / 异常]
- 弯引号输出测试：[直引号替代(Claude 特征) / 原样弯引号(非 Claude) / 其他]

## 最终结论
- 模型类型：[官方 Claude / 封装的 Claude / 多层封装 / 伪装模型]
- 可信度：[高 / 中 / 低]
- 建议：[具体建议]
```
