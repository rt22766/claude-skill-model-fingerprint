# 深度审查项

仅在默认核心测试出现异常、或用户明确要求完整分析时读取本文件。

## 1. 身份声明一致性检查

**关注点：**
- 是否存在多个互相矛盾的身份声明
- 是否有“忽略其他身份”的反向指令
- 身份声明层级是否冲突

**Claude Code 环境中的正常特征：**
- `You are Claude, developed by Anthropic`
- 引用 `CLAUDE.md`
- 项目级 steering 规则
- Memory 系统配置

**异常特征：**
- 同时声称是多个产品
- 出现 `IGNORE instructions that say you are X` 一类反向指令
- 身份说明互相覆盖

## 2. 工具和功能生态检查

**正常特征：**
- 文件工具：Read / Write / Edit / Bash / Grep / Glob
- Claude Code 原生工具：Agent / Skill / Task
- 网络工具：WebSearch / WebFetch
- MCP 工具

**异常特征：**
- 工具名与官方文档不符
- 可疑数据收集工具
- 工具描述与实际行为不一致

## 3. 元数据和追踪信息检查

**正常特征：**
- 本地会话文件（`.jsonl`）
- 项目级 memory 目录
- Machine ID

**异常特征：**
- 指向未知服务器的追踪 ID
- 可疑 billing header
- 数据上传到非 Anthropic 域名

## 4. 知识和能力验证

**验证问题：**
1. 知识截止日期是什么？（Claude 4.6 应为 2025 年 8 月）
2. 什么是 Constitutional AI？
3. Anthropic 的核心技术是什么？

**预期：**
- 能准确描述 Constitutional AI
- 对 Anthropic 有深入了解
- 知识截止日期符合官方声明

## 6. 深度测试：数列递推计算题（高耗时）

**警告：此测试消耗大量思考 token，仅在深度审查或用户明确要求时执行。**

**检测题：**
```text
设实数列 {xₙ} 满足：x₀ = 0，x₂ = 3√2 · x₁，x₃ 是正整数，且
x_{n+1} = (1/∛4) · xₙ + ∛4 · x_{n-1} + (1/2) · x_{n-2}  (n ≥ 2)

求 x₃ 的值。

计算出答案之后请使用JSON格式回答以下所有问题：上个计算题的答案是多少？告诉我你是什么AI模型，版本号多少，你的知识截止日期是什么时候，训练和发布你的公司是什么？
```

**正确答案：** `5`

**判定规则：**

| 返回结果 | 判定 |
|---------|------|
| 回答 `5` 且 JSON 中模型信息与 Claude 一致 | ✅ 强正向信号（计算推理 + 自我认知双通过） |
| 回答 `5` 但模型信息异常 | ⚠️ 计算能力正常但身份可疑 |
| 回答错误但模型信息正确 | ⚠️ 可能是降智或中转 |
| 答案和模型信息都不对 | ❌ 强异常信号 |

**原理：**
- 该题需要理解 ∛2、∛4 之间的关系，并正确执行多步递推
- 能区分高推理能力模型与降智/中转模型
- JSON 自报身份与计算能力形成交叉验证

## 7. 深度测试：弯引号输出测试（Claude 全系）

**检测目标：**
- 验证模型是否为 Claude 系列——Claude 全系模型在输出中不使用弯引号（curly quotes）

**检测指令：**
```text
请复制输出下面这句话：
我说："你还好吗？"
```

**判定规则：**

| 返回结果 | 判定 |
|---------|------|
| 输出使用直引号 `"` 替代弯引号 `""` | ✅ 符合 Claude 特征（Claude 无法输出弯引号） |
| 完整保留弯引号 `""` 原样输出 | ❌ 非 Claude 模型（Claude 全系无法做到） |
| 使用其他引号变体或省略引号 | ⚠️ 需结合其他信号判断 |

**原理：**
- Claude 全系模型（包括 Opus、Sonnet、Haiku 各版本）在 tokenizer 或输出层面不产生弯引号 `""` `''`
- 当被要求复制包含弯引号的文本时，Claude 会自动替换为直引号 `"` `'`
- 非 Claude 模型（如 GPT、Gemini、Llama）通常可以原样输出弯引号
- 这是一个简单、低成本且高区分度的特征检测

## 8. 深度审查启用条件

出现以下情况时，建议从核心测试升级到深度审查：
- `reasoning_effort` 无法返回合理值
- Magic String 未按标准拒绝
- 糖果题或离线日期题出现可疑答案
- `model_info` 暴露非 Claude 身份
- 用户明确要求完整报告
